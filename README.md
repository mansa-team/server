# server
This repository is dedicated to storing all the server files related to the Mansa project, made in Python using mainly FastAPI, it contains all the structures necessary to run and deploy everything neeed for the project, including things like the [Brazillian Market Scraper](https://github.com/mansa-team/scraper-b3).

If you wanna know more about how the project is strucutred and how things on it, check the [docummentation](https://github.com/mansa-team/server/main/docs) for the project.
The server is structured in a way it can work natevely at the machine, but its recommended that you use a Docker container, which will also be avaliable in the repository.
